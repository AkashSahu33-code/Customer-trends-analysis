# Customer Behavior Data Analytics Portfolio Project

## Overview

This project is a comprehensive end-to-end data analytics portfolio project designed to simulate real-world business analytics workflows followed by professional data analysts. The project demonstrates the complete analytics lifecycle, including data preparation, exploratory analysis, SQL-based business querying, interactive dashboard development, and insight-driven reporting.

The primary objective of the project is to transform raw customer and transactional data into meaningful business insights that support strategic decision-making and customer-focused business growth.


## Project Objectives

* Analyze customer purchasing behavior and market trends
* Identify key customer segments and purchase patterns
* Perform business-oriented data analysis using SQL
* Develop interactive dashboards for insight visualization
* Generate actionable recommendations through data-driven analysis



## Project Workflow

### Data Preparation & Exploratory Data Analysis (Python)

* Cleaned, transformed, and prepared raw datasets for analysis
* Performed feature engineering and exploratory data analysis (EDA)
* Identified customer trends, behavioral patterns, and data distributions using Python libraries

### Business Data Analysis (SQL)

* Executed SQL queries to analyze customer transactions, purchasing behavior, and business KPIs
* Performed aggregations, filtering, joins, and trend analysis to uncover actionable insights

### Dashboard Development & Visualization (Power BI)

* Built an interactive Power BI dashboard to visualize customer trends, sales performance, and key business metrics
* Designed dynamic visualizations and KPI-driven reports to support stakeholder decision-making

### Reporting & Business Insights

* Generated analytical reports summarizing key findings and customer behavior insights
* Presented business recommendations and data-driven strategies through structured reporting and visual storytelling
